# myDIY

Static website for organizing blue collar knowledge, hobbies, and how-to guides.

## Pages
- `index.html` – Landing page with overview and navigation.
- `knowledge.html` – Knowledge repository organized by trade.
- `hobbies.html` – Hobby Hub with interactive hobby tracker stored in your browser.
- `how-to.html` – Step-by-step instructions for common handyman tasks.

Open `index.html` in a browser or serve the root directory through any static web server.
